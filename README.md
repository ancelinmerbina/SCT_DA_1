# SCT_DA_1 - Excel Sales Dashboard

## Project Overview

This project was completed as part of the Data Analyst Internship at SkillCraft Technology. The objective was to analyze sales data using Microsoft Excel and create a dashboard to visualize key business insights.

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Data Cleaning
* Data Analysis

## Dashboard Features

* Total Sales KPI
* Sales by Category Analysis
* Monthly Sales Trend Analysis
* Interactive Pivot Charts

## Dataset

Sample Superstore Sales Dataset

## Key Learnings

* Data Cleaning and Preparation
* Creating Pivot Tables
* Building Pivot Charts
* Sales Trend Analysis
* Dashboard Design in Excel
